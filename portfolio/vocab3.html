HTML form

a section of a document containing normal content, markup, special elements called controls (checkboxes, radio buttons, menus, etc.), and labels on those controls.

HTML form action

specifies where to send the form-data when a form is submitted.

HTML form method

The method attribute specifies how to send form-data

HTML fieldset

The < fieldset > tag is used to group related elements in a form.

HTML legend

to provide a caption or a title for the associated < fieldset > element

HTML Input tag

specifies an input field where the user can enter data.

HTML attributesr

a piece of markup language used to adjust the behavior or display of an HTML element.

HTML checkboxes

aThe defines a checkbox.

HTML radio buttons

The defines a radio button.

HTML textarea

The < textarea > tag defines a multi-line text input control.

HTML button

a clickable button on a webpage that users can interact with.

HTML label

specifies the title of the text track.
