str/string

A string is a sequence of characters enclosed in either single quotes ('') or double quotes (“”).

int/interger

a whole number, positive or negative, without decimals, of unlimited length.

float

a function or reusable code in the Python programming language that converts values into floating point numbers.

list

a data structure in Python that is a mutable, or changeable, ordered sequence of elements.

tuple

a type of data structure that is very similar to lists.

Set

a data type in python used to store several items in a single variable.

dict/dictionary

a data structure, used to store values in key: value format.

boolean/bool

A Boolean value is either true or false.
