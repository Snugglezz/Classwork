Encapsulation-OOP

the bundling of data and the methods that act on that data such that access to that data is restricted from outside the bundle

Abstraction-OOP

hiding the complex implementation details and showing only the necessary features of an object.

instance

a unique copy of a blueprint.

JSON (JavaScript Object Notation)

a standard text-based format for representing structured data based on JavaScript object syntax

constructor

a special function that creates and initializes an object instance of a class.

Python self

used to refer to the current object instance within a method or function.

JavaScript this

refers to an object

imperative programming

your code is based on statements that change the program state by telling the computer how to do things.

declarative programming

a programming approach in which coders write code that describes what they want to do and what results they want to achieve, rather than how they will achieve the results.

Array.map()

a built-in method in JavaScript for arrays.

Polymorphism-OOP

enables a single function or method to exhibit different behaviors based on the context in which it is invoked.

model-view-controller design pattern

a design pattern that breaks an application into three parts: the data (Model), the presentation of that data to the user (View), and the actions taken on any user interaction (Controller).
