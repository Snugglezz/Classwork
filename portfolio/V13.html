Vocab 13

Uniform Resource Locator (URL)

the address of a web page

Firewall

a network security device that monitors and controls incoming and outgoing network traffic based on predetermined security rules, acting as a barrier between a trusted internal network and untrusted external networks.

Transmission Control Protocol-Internet Protocol (TCP-IP)

a suite of communication protocols that enables devices to communicate over networks, including the internet, by defining how data is packetized, addressed, transmitted, routed, and received.

Network Protocols

a set of rules and standards that govern how data is transmitted and exchanged between devices on a network, enabling them to communicate effectively, regardless of their underlying hardware or software.

Computer Network

the fourth version of the Internet Protocol, a set of rules that govern how data is transmitted over the internet, using a 32-bit address system.

IPv4

IPv6

the latest version of the Internet protocol, designed to address the issue of IPv4 address depletion by providing a much larger address space, enabling more devices to connect to the internet.

Domain Name Services (DNS)

a crucial part of the internet that translates human-readable domain names (like "www.example.com") into the numerical IP addresses that computers use to communicate.

Hyper-text transfer protocol (HTTP/HTTPS)

an application protocol for distributed, collaborative, hypermedia information systems that allows users to communicate data on the World Wide Web.

File Transfer Protocol

a standard network protocol used to transfer files between a client and a server over a TCP-based network, such as the internet.

Cloud Computing

the practice of using a network of remote servers hosted on the internet to store, manage, and process data, rather than a local server or a personal computer.

OSI Model

a conceptual framework that describes how data travels from one application to another across a network, breaking down network communication into seven distinct layers, each with specific functions.

Virtual Private Network (VPN)

a method employing encryption to provide secure access to a remote computer over the internet
