Logic Layer

The logic layer is the middle of the presentation and data access layer, and it helps the user with information its given.

layer

A layer can be a thing or a person, like a plant or a lazy person.

Synonyms

  • tier
  • surface

user interface

when both the human and computer communicate with eachother from a display screen, mouse...etc

software

the programs and applications on a computer/laptop

synonyms

  • program
  • application
  • code

application

its also called app, but it's something simular to a website but on a computer or a mobile device.

synonyms

user interface designer

a person that designs the computer that communicates with the user that interacts with it.

Front-end developer

a person that creates websites and web languages such as Python,HTML...etc

back-end developer

person or people who help/create the data access layer

data base developer

the create and test database systems and help users on the database.

Abstraction-abstract

its a abstraction summary of the core content that helps user with texts and passages.

pattern

its a repeating about of text in code, password, or anything input related.

ui layer

its a layer above the logic layer and it basicly shows what the user want to see or shows what the user is seeing.

persistence layer

another name for the UI layer, and it sits above the data access and the logic layer.

seperation of concerns

separating a space into rooms, so that activity in one room does not affect people in other rooms.

stack

a linear data structure, a collection of items of the same type.
