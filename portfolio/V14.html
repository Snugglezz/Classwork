Week 14 Vocab

Wireframe

a basic, two-dimensional visual representation of a webpage or app layout, focusing on structure, functionality, and user flow, without detailed visual elements like colors or images.

User story

a concise, user-centric description of a desired feature or functionality

mood board

a visual tool, like a digital collage, that helps designers and clients communicate the desired aesthetic, style, and overall feeling of a website, using images, colors, typography, and other design elements.

Richard Saul Wurman's LATCH

a framework for organizing information, standing for Location, Alphabet, Time, Category, and Hierarchy

sans-serif and serif fonts

serif fonts feature small lines or strokes at the ends of letter strokes, while sans-serif fonts lack these strokes

Usability

the ease with which a user can navigate, interact with, and achieve their goals on a website

Sitemaps for Google

a file that lists all the pages of a website

Sitemaps visual

a diagrammatic representation of a website's structure, showcasing the hierarchy of pages and their relationships, acting as a blueprint for planning and communication

story board

a diagrammatic representation of a website's structure, showcasing the hierarchy of pages and their relationships, acting as a blueprint for planning and communication

Static and Variable web fonts

static web fonts use separate files for each font style (e.g., regular, bold, italic), while variable web fonts store multiple styles within a single file, allowing for dynamic font variations and smaller file sizes

Card sort

a UX research technique where participants sort cards representing content or pages into groups based on how they perceive the information

form tag with action and method attributes

The action attribute defines the location (URL) where the form's collected data should be sent when it is submitted
