Week 15 Vocab

Database Join Many to Many

one record can be associated with many other records and vice-versa.

Database Report

a document that organizes and presents data extracted from a database for specific purposes.

Database Table

each table stores data about a particular subject, such as employees or products.

Database join (1:1,1:Many)

a command clause that combines records from two or more tables in a database.

Structured Query Language (SQL)

a standardized programming language that is used to manage relational databases and perform various operations on the data in them.

Database/Database Management System

a software system that allows users to create, define, manipulate and manage databases.

NoSQL Database

use a variety of data models for accessing and managing data.

Database Table Primary Key

a column in a relational database table that's distinctive for each record.

Business Intelligence and Business Analytics

Business Intelligence (BI) focuses on analyzing past and present data to understand current business operations, while Business Analytics (BA) uses historical data and statistical methods to predict future trends and outcomes.

Data Normalization/Denormalization

Normalization reduces redundancy by splitting data into multiple related tables.

Database Table Foreign Key

a column or columns of data in one table that refers to the unique data values -- often the primary key data -- in another table
