Vocabulary 9

Standard 3: Fundamental Skills for IT Careers

Students will be able to demonstrate mastery of concepts fundamental to careers in information technology, including basic hardware and networking technologies, common operating systems and general data management, and cybersecurity fundamentals.

Exposed Competent Proficient Advanced
Student can recall the basic idea of the standard. Student can explain the standard correctly. Student can apply or demonstrate the standard correctly. Student can evaluate the strengths and weaknesses of the standard and conclude when to use it.

3.a Research and identify Cybersecurity topics

3.b Identify security principles, vulnerability, and threats in software

3.c Identify security principles, vulnerability, and threats in network infrastructure

3.d Analyze threat maps to identify cyberattack targets and origins.

3.e Explain what secure systems are from a Confidentiality, Integrity, availability (CIA triad) perspective.

3.f Research and identify components of access control such as Identification, authentication, and authorization

3.g Identify and demonstrate the basics of cryptography and research and describe topics in cryptology, cryptography, and cryptanalysis

3.h Research and describe penetration testing methods in the context of ethical hacking.

3.i Classify hardware components, their functions, and relationships and analyze different hardware components required for software applications for a specific use.

3.j Demonstrate programming a physical computing device.

3.k Explain concepts fundamental to networking.

3.l Describe the purpose of a network and identify and explain the functions of network hardware components.

3.m Explain cloud computing and its components. Explain how network topologies and protocols enable users, devices, and systems to communicate with each other.

3.n Research and identify server-side networking software components. (Firewalls, Domain Name System (DNS), Visual Private Network (VPN), Proxy Servers, Dynamic Host Configuration Protocol (DHCP), File Transfer Protocol (FTP), Data Center (DC), and Active Directory (AD)).

3.o Identify security risks associated with storing information on network devices.

3.p Describe the purpose of an operating system.

3.q Differentiate between desktop, server, and mobile device operating systems.

3.r Differentiate between a Command Line Interface and a Graphical User Interface.

3.s Demonstrate the ability to navigate a file system at the Command Line Interface.

3.t Identify the content associated with different file types and why different file types exist (e.g., formats for word processing, images, video, sound, and three-dimensional objects).

3.u Research and compare data compression algorithms and their applicability to different contexts.

3.v Research and identify data management concepts and explain the purpose of data management systems.

3.w Demonstrate the planning, designing and creation and use of a relational database.

3.x Research and describe the relationship between tables such as one-to-one and one-to-many.

3.y Explain the purpose of data analysis.
