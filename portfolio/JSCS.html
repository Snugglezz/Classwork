javascript Cheat Sheet

let

its a statement in javascript that has a variable that cant change

const

a javascript statement that has a changeable variable

for loop

its used to repeat the same codes for a fixed amount of times (yk how many times it will repeat)

Array.forEach()

A function to execute for each element in the array. Its return value is discarded.

document.getElementById()

returns an element with a specified value.

document.getElementByTagName()

returns a collection of all elements with a specified tag name.

document.getElementByClassName()

returns a collection of all elements with a specified class name.

document.querySelector()

returns the first Element within the document that matches the specified CSS selector, or group of CSS selectors.

document.querySelectorAll()

A string containing one or more selectors to match. This string must be a valid CSS selector string; if it isn't, a SyntaxError exception is thrown.
